## Quick Start

Git clone

```bash
git clone https://github.com/babylon-finance/protocol.git
```

Install dependencies

```bash
yarn
```

Run

```bash
yarn compile
```

to compile Solidity contracts.

Run

```bash
yarn generate
```

to generate mnemonic.txt file.

## Gas Report

Run

```bash
yarn gas-report
```

## Coverage

Run

```bash
yarn coverage
```

## Test

Run

```bash
yarn test
```
